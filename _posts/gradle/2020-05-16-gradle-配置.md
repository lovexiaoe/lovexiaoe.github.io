### Configuration
一个Configuration代表了一组构建和他们的依赖，在gradle中是依赖管理的一部分。声明和设置一个Configuration如下：
```groovy
apply plugin: 'java' 

configurations {
  //添加一个 configuration
  myConfiguration

  //添加一个 configuration，并让它继承自testImplementation
  myIntegrationTestsCompile.extendsFrom(testImplementation)

  //禁止传递依赖
  compileClasspath.transitive = false
  testCompileClasspath.transitive = false
}
```
