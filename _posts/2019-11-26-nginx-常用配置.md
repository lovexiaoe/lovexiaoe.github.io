## 禁用ip访问
使用default_server和_实现禁用ip访问，如下配置达到禁用80端口ip访问的效果。

```
server {
    listen       80 default_server;
    listen       [::]:80 default_server;
    server_name  _;
    return 444;
}

server {
    listen       80;
    listen       [::]:80;
    server_name  www.xxx.cn;
    rewrite ^(.*) https://$host$1 permanent;
}
```
